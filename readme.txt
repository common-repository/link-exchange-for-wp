== Installation ==


1. Upload `links_exchange_free.php` and the folder 'styles' to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Adjust any of the setting under the LinkExchange folder created by the plug-in
4. Configure any options as desired, and then enable the plugin
5. That's it!
